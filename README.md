# GoWiki

Projeto simples de wiki feito com a linguagem Go
## Como usar

Para rodar este projeto você precisa ter o Go instalado, as instruções para instalação desse projeto podem ser encontradas em:  
https://golang.org/dl/

este projeto funciona utilizando um navegador web, a porta padrão é a porta 3000, ao baixar o projeto deve ser criada na raiz do projeto uma pasta chamada files, onde serão salvos os arquivos de texto, e dos quais serão lidos. o objetivo deste projeto é ser simples e útil. além de servir como aprendizado da linguagem GO  
Atualmente há duas páginas, a de edição, e de visualização dos textos.
* Na pasta raiz do projeto após ter criado a pasta files execute o comando go run .
* Para visualizar um texto acess http://localhost:3000/view/nome-texto  
* Para editar ou criar um arquivo de texto http://localhost:300/edit/nome-texto
